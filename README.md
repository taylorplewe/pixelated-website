I was wondering how one might go about making a website pixelated. Turns out the new [HTML-in-Canvas API](https://yashrajbharti.github.io/html-in-canvas/) was exactly what I was looking for.

It would look a lot better if I could disable anti-aliasing. Still looking for that solution.

[https://tplewe.com/pixel-site](https://tplewe.com/pixel-site)

> [!IMPORTANT]
> For the site to work, you MUST be on [Chrome Canary](https://www.google.com/chrome/canary/) and enable the "HTML-in-Canvas" feature flag (`chrome://flags/#canvas-draw-element`).
